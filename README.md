# Portfolio de Traduction de Webtoons

**Site portfolio présentant** mes *traductions professionnelles* de webtoons du coréen vers le français, réalisées entre 2020 et 2023.

## À Propos

Ce portfolio présente mon **travail de traduction** pour diverses plateformes :
- **3 années** d'expérience professionnelle
- Collaboration avec **plusieurs clients et plateformes**
- Œuvres disponibles sur *Delitoon*, *Ono* et autres plateformes
- **Traduction du coréen vers le français**

## Structure du Site

### Pages Disponibles :
- `index.html` - **Page d'accueil** avec présentation
- `MyPortfolioWebtoons.html` - **Catalogue des webtoons** traduits
- `MyPortfolioClients.html` - **Clients et plateformes**  
- `MyPortfolioContact.html` - **Contact professionnel**

### Technologies Utilisées :
- `HTML` - Structure et contenu du site
- `CSS` - Mise en page et style visuel
- `XML/XSLT` - Gestion des données 
- `GitHub Pages` - Hébergement du site
